# 🏛️ Government College of Engineering Thanjavur — College Website

A responsive multi-page college website built with pure **HTML, CSS, and JavaScript** as a mini project. The site showcases the CSE department of GCE Thanjavur with pages for academics, faculty, events, and contact information.

---

## 🔗 Live Demo

> Deploy on GitHub Pages — go to **Settings → Pages → Source → main branch / root** and your site will be live at https://github.com/joelfranklin22/college-website

## 📁 Project Structure

```
├── index.html          # Home page — about college, news ticker, placement tables
├── Academic.html       # Academics — Vision, PEO, PO, PSO, Facilities
├── staffs.html         # Faculty — table with photos and designations
├── Events.html         # Events — Naan Mudhalvan, Hackathon, Industrial Visit
├── contact.html        # Contact — staff directory and innovation council
├── CSS/                # stylesheet (fully responsive)
├── JS/                 # Javascript (Validation,Animation,Api,Event Handling)
└── assets/Images       # Images — college photos, logos, faculty photos
```

---

## ✨ Features

- **Fully Responsive** — works on mobile, tablet, and desktop
- **Hamburger Navigation** — collapsible menu on small screens
- **Image Slideshows** — auto-cycling galleries on the Events page
- **News Ticker** — animated scrolling news & updates section
- **Placement Tables** — on-campus and off-campus placement records
- **Faculty Directory** — table with circular photo frames and designations
- **Contact Directory** — staff contacts and Accreditation & Innovation Council
- **Animated Footer** — 4-column responsive grid with social media links
- **Google Fonts** — Playfair Display and DM Serif Display

## 🚀 Getting Started

### Run Locally

1. Clone the repository:
   ```bash
   https://github.com/joelfranklin22/college-website/
   ```
2. Open the project folder:
   ```bash
   cd <repo-name>
   ```
3. Open `index.html` in your browser — no build step needed!

### Deploy on GitHub Pages

1. Push the project to a GitHub repository
2. Go to **Settings → Pages**
3. Set Source to **Deploy from a branch → main → / (root)**
4. Your site will be live in a few minutes

## 🎓 About the College

**Government College of Engineering, Sengipatti, Thanjavur** was established in 2013 by the Government of Tamil Nadu. It is approved by AICTE, New Delhi and affiliated to Anna University, Chennai. The college spans 35.73 acres and offers B.E. programs in CSE, Civil, EEE, ECE, and Mechanical Engineering.
